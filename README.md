
#  Finlatics Data Analytics Projects

##  About this repository

This repository contains the projects I completed as part of my Finlatics certification. These projects helped me understand how data analysis and business thinking come together to solve real-world problems.

Each project focuses on a different skill such as business analysis, Excel, and Power BI.



## Projects Included

###  Project 1 – Business Analysis (IT Company Case Study)

In this project, I analyzed why an IT company was not improving its profit margins as much as its competitors.

* Identified the core problem: low margin growth despite good revenue
* Broke down profit into revenue and cost components
* Analyzed sector-wise and geography-wise performance
* Suggested improvements like focusing on high-margin sectors (healthcare, BFSI), reducing contractor costs, and investing in technologies like AI and cybersecurity

 This project helped me understand how companies improve profitability, not just revenue.



###  Project 2 – Power BI Dashboard

In this project, I created a dashboard to visualize business data.

* Built interactive visuals
* Used charts to show trends and comparisons
* Focused on making insights easy to understand

 This improved my data visualization and storytelling skills.



### Project 3 – Excel Analysis

This project involved working with data in Excel.

* Cleaned and organized data
* Used formulas and basic analysis
* Extracted insights from raw data

 Helped me build strong fundamentals in Excel.



###  Project 4 – Advanced Excel Analysis

This project focused on deeper analysis using Excel.

* Applied more advanced calculations
* Compared different variables
* Generated meaningful insights

 Strengthened my ability to work with structured data.



##  Tools & Skills Used

##  Key Learnings

* Data alone is not enough — interpreting it is important
* Business understanding is as important as technical skills
